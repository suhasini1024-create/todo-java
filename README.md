## üìù Commands

| Command | Description |
|--------|-------------|
| `add <text>` | Add a new todo |
| `list` | Show all todos |
| `done <id>` | Mark todo as done |
| `rm <id>` | Remove todo |
| `edit <id> <text>` | Edit an existing todo |
| `clear` | Remove all todos |
| `help` | Show help menu |
| `exit` | Quit the app |
